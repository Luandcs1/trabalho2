# Sistema de Itinerário

Projeto em C++ utilizando lista encadeada para simular rotas de ônibus.

## Estrutura

- include → cabeçalhos (.h)
- src → implementação (.cpp)
- test → testes
- main.cpp → menu principal

## Estrutura de dados

Lista simplesmente encadeada.

## Justificativa

Estrutura simples, eficiente para inserções e remoções.

## Cenário alternativo

Lista circular seria ideal para linhas circulares.
